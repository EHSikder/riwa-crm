## 🎬 Recording Setup

### Equipment Needed
- Computer with screen recording software
- Microphone (built-in or external)
- Quiet environment
- Good lighting (for webcam if showing face)

### Recommended Software

**Mac:**
- QuickTime Player (built-in)
- ScreenFlow
- OBS Studio (free)

**Windows:**
- OBS Studio (free)
- Camtasia
- ScreenFlow alternative

**Linux:**
- OBS Studio (free)
- SimpleScreenRecorder

### Settings

**Resolution:** 1920x1080 (Full HD)
**Frame Rate:** 30 fps
**Audio:** 44.1 kHz, Stereo
**Format:** MP4 or WebM

## 📸 Screen Recording Tips

### 1. Preparation
- Close unnecessary applications
- Clear desktop
- Maximize browser window
- Use light theme for better visibility
- Zoom in on UI (125-150%)

### 2. Recording
- Speak clearly and slowly
- Pause between sections
- Click deliberately
- Move mouse smoothly
- Avoid rapid movements

### 3. Pacing
- 2-3 seconds per action
- 3-5 seconds for reading text
- 1-2 seconds between sections

## 🎙️ Audio Recording

### Voice-Over Tips
- Use a quiet room
- Speak into microphone at 6 inches
- Use a script but sound natural
- Record in sections
- Do multiple takes

### Audio Levels
- Peak at -6dB
- Average around -18dB
- No clipping or distortion

## ✂️ Video Editing

### Basic Editing
1. Trim unnecessary parts
2. Add intro/outro
3. Add captions
4. Add background music (optional)
5. Color correction

### Tools
- DaVinci Resolve (free)
- Adobe Premiere Pro
- Final Cut Pro
- iMovie (Mac)

### Captions
- Add for accessibility
- Highlight key points
- Use consistent font
- Timing: 2-3 seconds per caption
